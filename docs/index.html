What is this?

This is Jekyll Scout, a Jekyll front-end for Help Scout Docs. Use Help Scout like a headless CMS and take advantage of the flexability of Jekyll and the awesome features of Help Scout.

View on GitHub

Advantages over plain old Jekyll

  1. Articles available on Help Scout Beacon
  2. Articles available in Help Scout for linking in ticket replies

Advantages over Help Scout hosted docs

  1. Completely customizable themes
  2. Host anywhere you want
  3. Use a custom domain with SSL

Disadvantages

  1. Site needs rebuilding on content changes (can be automated)
